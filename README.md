# Penomato MVP

## ?? Descrição
MVP do sistema Penomato para gestão.

## ?? Tecnologias
- PHP 7.4+
- MySQL/MariaDB
- Bootstrap 5
- Git

## ? Instalação Rápida

### Pré-requisitos
- XAMPP/WAMP instalado
- Git instalado
- PHP 7.4+

### Passos
1. Clone o repositório:
\\\ash
git clone https://github.com/niltonbakargi/penomato_mvp.git
\\\

2. Configure o banco de dados em \config/database.php\

3. Acesse via navegador: \http://localhost/penomato_mvp/public/\

## ?? Estrutura de Pastas
- \src/\ - Código fonte principal
- \public/\ - Arquivos acessíveis via web
- \config/\ - Configurações da aplicação
- \database/\ - Migrações e seeds
- \	ests/\ - Testes automatizados
- \docs/\ - Documentação
- \scripts/\ - Scripts auxiliares
